# Dictionary-App
Created a Dictionary App using HTML, CSS3, JavaScript, Free Dictionary API
In this Application user can get:
1. Word Meaning
2. Parts of Speech
3. Antonyms
4. Synonyms
5. Example
6. Read More Button to know more about the Word on Wikipedia
